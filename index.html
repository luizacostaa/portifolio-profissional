Filosofia Criativa

Minha visão sobre arte, tecnologia e sua fusão infinita

"A criatividade não é exclusiva dos artistas. Todo desenvolvedor é um artista, toda mulher em tech é uma visionária que pinta o futuro com código."

Convergência de Mundos

Para mim, arte e tecnologia não são opostos - são complementos perfeitos. Quando programo, estou criando estrutura. Quando pinto, estou criando emoção. Quando combino ambas, crio magia.

Expressão Autêntica

Acredito que cada pessoa tem uma voz única, uma perspectiva única. Minha missão é usar tecnologia para amplificar essas vozes, especialmente as das mulheres, para criar representatividade genuína.

Liberdade Criativa

Sem limitações técnicas, somos livres para sonhar. E sem limitações sociais, mulheres são livres para criar. Isso é o que eu busco em cada projeto.
